# MusicApp - (DJ Musicz)
#### Description
```
This Project is an online music app. Users can listen songs online.
```
#### Features

```
1. New User Registration
2. User Login
3. Forgot Password For Users
4. Admin Login
5. Admin (add) [Languages, Movies, Songs, Genre And Genre Songs]
6. Admin (Delete) [Languages, Movies, Songs, Genre And Genre Songs]
7. Play Song Online By Searching From Languages-> Movies-> Songs
8. Search Song By Song Name
9. Playlist (add songs)
10. Playlist (remove songs and Clear Playlist)
11. Playlist with audio control
12. Search Song By Movie Name
13. Music Genre Recommendation Based on Age
14. Recently Released Movies List based on Movie Release Date
15. Kids Playlist
16. Displaying List Of All Available Movies
```
